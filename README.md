# 🔥 Sentiment Analysis App

This is a simple web app that analyzes text sentiment using Python Flask.

## 🚀 Features

* Enter any text
* Get sentiment result (Positive / Negative / Neutral)

## 🛠️ Tech Stack

* Python (Flask)
* HTML / CSS
* TextBlob (NLP)

## ▶️ How to Run

1. Install requirements:
   pip install -r requirements.txt

2. Run app:
   python app.py

3. Open in browser:
   http://127.0.0.1:5000
